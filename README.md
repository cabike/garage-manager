## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/garage-manager.git
cd garage-manager

# Install dependencies
npm install

# Start development server
npm run dev
```

Open http://localhost:3000 in your browser.
